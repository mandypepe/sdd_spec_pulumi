# Data Model: K8s Base Infrastructure

## Entities

### OrchestratorCluster
Represents the core Kubernetes control plane infrastructure.
- `name` (String): Unique identifier.
- `az_list` (List[String]): Availability zones for distribution.
- `control_plane_config` (Object): Configuration for HA control plane.

### ComputeNodePool
Represents a collection of private compute instances.
- `name` (String): Unique identifier.
- `min_size` (Integer): Minimum node count.
- `max_size` (Integer): Maximum node count.
- `instance_type` (String): Node instance specifications.
- `network_config` (Object): Private network segmentation.

### SecurityLogBucket
Represents the WORM storage for security logs.
- `bucket_name` (String): Bucket identifier.
- `retention_days` (Integer): 90 days.
- `compliance_mode` (Boolean): True (WORM).

## Relationships
- `OrchestratorCluster` distributes resources across multiple `az_list` items.
- `ComputeNodePool` is associated with an `OrchestratorCluster` and resides in defined subnets.
- `SecurityLogBucket` stores logs generated by the `OrchestratorCluster` and `ComputeNodePool` activities.
